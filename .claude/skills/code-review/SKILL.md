## 코드 리뷰 스킬

`code-reviewer` subagent를 `background: true`로 spawn한다.

에이전트는 자체적으로 git diff를 수집하고 체크리스트를 기반으로 리뷰를 수행한다. 스킬은 spawn만 담당한다.

사용자에게는 즉시 "리뷰가 시작되었습니다. 백그라운드에서 진행 중이며 완료되면 결과를 표시합니다." 메시지를 출력한다.
