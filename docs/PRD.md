# PRD: 도트 북 다이어리

## 목표
읽은 책·문장·독후감을 "도트 그림체의 따뜻한 방"이라는 공간감 있는 UI에서 기록하고 회고하게 한다. 실용성만 있는 독서 기록 앱 대신, 일상적으로 다시 열고 싶은 정서적 경험을 만드는 것이 핵심이다.

## 사용자
- **회원**: 여러 기기에서 독서 기록을 이어가고 싶은 독자. Supabase 계정으로 로그인.
- **비회원**: 가입 없이 바로 써보고 싶은 독자. 기기 로컬(IndexedDB)에만 저장되며, 데이터는 해당 기기에 국한된다.

## 핵심 기능
1. **도트 방 메인(`/`)** — 곰이 러그에 앉아 책을 읽는 방. 다이어리·책더미·창문·톱니에 hitbox를 두어 각 페이지로 이동한다.
2. **책 등록(`/add-book`)** — 바코드 스캔(@zxing/browser) 또는 제목/출판사/ISBN 검색(알라딘 Open API). 결과를 선택해 내 책장에 추가.
3. **책장(`/bookshelf`)** — 등록한 책을 표지 썸네일 그리드로 본다. 표지 클릭 → `/reading/[bookId]`.
4. **독서 세션(`/reading/[bookId]`)** — 해당 책의 독서 시간, 현재 페이지, 관련 문장/독후감 기록.
5. **다이어리(`/diary`, `/diary/new`, `/diary/[id]`)** — 문장 발췌(quote) 또는 독후감(review) 엔트리 리스트/작성/편집.
6. **책 캘린더(`/book-calendar`)** — 월간 grid. 날짜 셀에 그날 읽은 책 표지 미리보기.
7. **설정(`/settings`)** — 닉네임, 로그인/로그아웃, (v1.1) 로컬→서버 동기화 토글.
8. **로그인(`/login`)** — Supabase Auth(이메일 + Google).

## MVP 제외 사항
- 소셜 기능(공유·팔로우·피드)
- 비회원 → 서버 데이터 동기화 (v1.1에서 명시 동의 후 업로드)
- 통계/그래프 대시보드
- 푸시 알림
- 다국어 (한국어만)
- 커스텀 테마 에디터 (라이트/다크 토글만)

## 디자인
- **도트 그림체 일관성 유지**: 정적 PNG 스프라이트(방/곰/다이어리/책더미/창문/램프) 레이어 합성. `image-rendering: pixelated`로 업스케일.
- **폰트**: Galmuri11(한글 픽셀, OFL) 셀프호스팅. 안티앨리어싱 off.
- **색상**: 따뜻한 실내 톤. 배경 `#2a1f17`(밤) / `#f4e4c1`(낮 크림), 포인트 `#e89b5e`(램프 오렌지).
- **금지**: `rounded-*`, `backdrop-filter`, `gradient`, box-shadow blur, 보라/인디고 브랜드 색 — 모던 SaaS처럼 보이면 실패.
